    SimpleCanvas Library

    Overview

    The SimpleCanvas library is a lightweight and easy-to-use JavaScript library designed for creating and managing canvas-based graphics and animations. It provides a simple API for handling drawing, event management, and animations, making it an excellent choice for beginners and educators in game development.

    Features

    • GameCanvas Class: A central class for setting up a canvas and managing drawings and events.
    • Drawing Management: Easily add, remove, and restore drawings on the canvas.
    • Event Handling: Built-in methods for handling mouse events, including clicks and movements.
    • Canvas Resizing: Automatically adjusts the canvas size based on the DOM element's size.
    • Animation Support: Simple methods for creating animations with elapsed time tracking.

    Installation

    To use the SimpleCanvas library, include the SimpleCanvasLibrary.ts file in your project. If you are using TypeScript, ensure that your project is set up to compile TypeScript files.

    Usage

    Basic Example

    import { GameCanvas } from './SimpleCanvasLibrary';

    const game = new GameCanvas('myCanvasId');
    game.addDrawing(({ ctx, elapsed }) => {
        ctx.fillStyle = 'blue';
        ctx.fillRect(50, 50, 100, 100);
    });
    game.run();

    Adding Click Handlers

    game.addClickHandler(({ x, y }) => {
        console.log(`Canvas clicked at: ${x}, ${y}`);
    });

    Type Definitions

    The library includes TypeScript type definitions to enhance the development experience. The Size type is defined in src/types/index.ts and represents the dimensions of the canvas.
